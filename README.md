# -FHW-Webcam
## -Definición
La Webcam o cámara de red es un periférico de entrada que captura imágenes y vídeo del mundo real para convertirlos en datos digitales. Estos datos pueden ser procesados por una computadora, transmitidos en tiempo real a través de Internet o almacenados como archivos multimedia.
